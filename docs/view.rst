View, Node & Connection
=======================

This module contains the visual-representation layer of pyArchimate:

* ``View`` – a named diagram that belongs to a :doc:`model`.
* ``Node`` – the visual representation of an :doc:`element` inside a ``View``.
* ``Connection`` – the visual representation of a :doc:`relationship` inside a ``View``.
* ``Profile``, ``Point``, ``Position`` – supporting geometry types.
