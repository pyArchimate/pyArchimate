# Quickstart: AI-Optimized Documentation

## Overview
Automate `AI.md` generation and verify tutorial snippets.

## Generate Documentation
1. Set `ANTHROPIC_API_KEY`.
2. Run: `python3 scripts/generate_ai_docs.py`
3. Check `AI.md`.

## Verify Tutorial
1. Run: `pytest tests/integration/test_tutorial.py`
